Encodes the BIP341 signing data for any flag type into a given object implementing the [io::Write] trait. This version takes a pre-computed annex hash and panics on error. Code mostly taken from: https://github.com/rust-bitcoin/rust-bitcoin/blob/9782fa8412e1c767998d018f6c915e51553a83d6/bitcoin/src/crypto/sighash.rs#L619
