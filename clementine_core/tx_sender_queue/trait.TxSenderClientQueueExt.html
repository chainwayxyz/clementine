where Self: 'async_trait, 'life0: 'async_trait, 'life1: 'async_trait, 'life2: 'async_trait, 'life3: 'async_trait, 'life4: 'async_trait,

Adds a transaction to the txsender sending queue based on core transaction semantics.

This function is a core-level wrapper around [TxSenderClient::insert_try_to_send]. It determines the appropriate FeePayingType and any cancellation/activation dependencies based on the [TransactionType].

IMPORTANT: insert_try_to_send is transactional. This helper requires an active DB transaction and will not partially insert state.
